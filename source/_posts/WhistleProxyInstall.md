# 實作
## 使用Node安裝Whistle
```bash
npm i -g whistle
```

<!--more-->
## 啟動Whistle
```bash
w2 start
```

## 查看Whistle狀態
```bash
w2 status
```

## 停止Whistle
```bash
w2 stop
```
