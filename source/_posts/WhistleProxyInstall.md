# 使用 Node 安裝 Whistle
```bash
npm i -g whistle
```

<!--more-->

# 啟動 Whistle
```bash
w2 start
```

# 查看 Whistle 狀態
```bash
w2 status
```

# 停止 Whistle
```bash
w2 stop
```
